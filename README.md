# MultiGit

View multiple git histories in one dashboard.

## Install

```sh
git clone https://github.com/jseashell/multigit
cd multigit

# optional, MultiGit requires Node v18+
nvm install

npm install
```

## Running the App

```sh
# development mode
`npm run dev`

# production build
`npm run build`
```
